# Desafio 14 – Tendências em Linguagens de Programação

## 🎯 Objetivo

Analisar criticamente uma linguagem emergente, suas vantagens, desvantagens e o contexto em que ela tem se destacado.

## 💡 Linguagem Escolhida: Rust

### 🔍 Características

- Segurança de memória sem uso de garbage collector
- Performance comparável ao C/C++
- Gerência de concorrência segura em tempo de compilação
- Ownership e borrowing para controle de acesso à memória

### ✅ Pontos Fortes

- Sem *null*, *data race*, *segfault*
- Forte sistema de tipos
- Comunidade ativa e bem documentada

### ⚠️ Desafios

- Curva de aprendizado íngreme
- Compilação mais lenta que linguagens mais maduras
- Ainda consolidando no mercado de trabalho

### 💬 Comparação

| Linguagem | Segurança | Performance | Concorrência | Aprendizado |
|-----------|-----------|-------------|--------------|-------------|
| Rust      | Excelente | Alta        | Segura       | Difícil     |
| C++       | Média     | Alta        | Arriscada    | Médio       |
| Go        | Boa       | Média       | Fácil        | Fácil       |
